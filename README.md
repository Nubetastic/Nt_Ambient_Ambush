# Nt_Ambient_Ambush


### No dependencies required!


## Features:
- Configurable Ambush regions and sub-regions.
- Configurable Safe regions and radius based coords.
- Configurable Enemy/weapon tables per region.
    - Configurable weapon chance per region per weapon type.
- Several Exports for ambush utilization in other scripts.
- Built in tools for script configuration.

## Description

**Standalone** Redm Script to add random enemy ambush encounters outside of towns and other safe zones.

Easily configurable safe areas, using Zone and Region or Coords and Radius.

Ambush spawn with custom enemy groups, with their own weapon tables.

Ambushes spawn using a set math.random() range for the base amount, and then adds additional math.random() every other player in the ambush zone.

## Exports

There are several exports, detailed in Docs/Exports.md, that can be used by other scripts to utilize ambushes to spawn enemies for their own missions.

## Tools Needed
Utility tools are built into this script so no third party script is needed to get the information you need.
